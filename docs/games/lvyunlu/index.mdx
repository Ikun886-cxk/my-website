## <i class="fa-duotone fa-badminton"></i> 游戏介绍

### <i class="fa-duotone fa-cloud-meatball"></i> 故事背景
  
<Typography variant="subtitle2" gutterBottom component="div">
	  分封制王国“万衍”王权渐衰，各封国间烽烟四起。东海边小国“帛”十一年前被北方大国“戮”所灭。十一年后，帛故都宛央城郊，帛太子荒朱带领援军前来，准备夺回家国。 战棋游戏《履云录》中，玩家将作为指挥官，与八位主角一起见证东海各国的兴衰。
	  </Typography>

### <i class="fa-duotone fa-fire-flame"></i> 游戏特色

<div class="max-w-md mx-auto rounded-xl shadow-md overflow-hidden md:max-w-2xl">
		<Fancybox data-fancybox data-src="https://onedrive-vercel-index-beta-pearl.vercel.app/api/raw/?path=/wiki/%E5%B1%A5%E4%BA%91%E5%BD%95%E6%B8%B8%E6%88%8F%E7%94%BB%E9%9D%A2-x4.webp">
          <img class="rounded-t-lg" src="https://onedrive-vercel-index-beta-pearl.vercel.app/api/raw/?path=/wiki/%E5%B1%A5%E4%BA%91%E5%BD%95%E6%B8%B8%E6%88%8F%E7%94%BB%E9%9D%A2.webp"/>
		  </Fancybox>
  </div>  
<Typography variant="subtitle2" gutterBottom component="div">
	  《履云录》中，玩家将扮演东海边小国——“帛”的初阶指挥官，于山雨欲来、乌云压城之时通过巧妙地排兵布阵和高超的指挥来帮助刚刚复国的帛国抵挡环伺的强敌的侵攻，一步一步探求各方势力的所求并最终做出抉择将故事引向不同的方向。
	  游戏采用半即时的“时刻”制。在行动点数耗尽前，角色可以任意移动和使用技能；但消耗的行动点数越多，下一次行动的到来就会变得越慢。仔细注意行动次序条，集中力量歼灭敌人吧。
	  通过不同的主动技能和被动技能的搭配，同一位队友可以以完全不同的定位出场。
	  通过完美破关、章间休息的活动、完成支线以及各种决断招募新的队友并提升与队友的默契度以获得各种好处。
	  在帛国境内四处活动、收集数十种宝物来强化己方角色。重要物品可能会影响结局走向哟。
	  </Typography>	  

### <i class="fa-duotone fa-address-card"></i> 关于万衍世界	  
	
<Typography variant="subtitle2" gutterBottom component="div">
		  在万衍大陆，一个传国数千年的分封制国家“万衍国”在享用了长久的和平后还是不可避免地走到了分崩离析的边缘。宗主国——万衍威权从八百年前就逐渐衰落，各个分封国之间互相征伐，战火吞灭了不计其数的国家和生命。
		  传统军事强国——戮，百年间不断兼并周围国家，现在已是万衍境内首屈一指的强大势力。新近成立的七国联盟则在盟主洪国的领导下与戮国对立，并极尽所能拉拢更多的国家加入联盟。两大势力的夹缝中则是游戏的主要舞台——帛国。
		  帛国于十一年前被戮国突袭攻灭，国王奎于王宫——宛央宫中举火殉国，大王子荒朱携三王子出奔南方的洪国。此后旧帛国领便为戮国占据。十一年后，为光复国家，大王子荒朱趁占领军松懈不备领大军杀回旧帛国领地，战争和死亡的阴影再次笼罩在东海之滨。多方势力也各怀鬼胎不约而至，这一场复国战争也许只是一系列混乱的开始……
	  </Typography>
